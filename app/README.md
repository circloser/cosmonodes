# Cosmonodes ✦

나를 중심으로 인간관계를 **별자리(force-directed 그래프)**처럼 시각화하는 절제된 SNS의 동작 데모입니다.

- 노드(별) = 내가 만든 **비공개 연락처**. 상대가 가입·수락하면 매칭됩니다.
- 매칭되면 그 사람의 1차 인맥이 **'희미한 먼 별'(2-hop)**로 내 우주에 확장됩니다.
- 별을 추가하면 **supernova 애니메이션**이 재생됩니다.

> 이 데모는 **백엔드 계정 없이** 브라우저에서 바로 동작합니다(데이터는 브라우저 localStorage에 저장).
> 실제 로그인/매칭 서버 연결은 `docs/SUPABASE_SETUP.md`를 참고하세요.

## 실행 방법 (비개발자용)

1. [Node.js](https://nodejs.org) 가 설치되어 있어야 합니다.
2. 터미널에서 이 `app` 폴더로 이동합니다.
3. 처음 한 번만 의존성을 설치합니다:
   ```
   npm install
   ```
4. 개발 서버를 켭니다:
   ```
   npm run dev
   ```
5. 터미널에 표시되는 주소(보통 `http://localhost:5173`)를 브라우저에서 엽니다.

## 사용해 보기

- **별 추가**: 가운데 아래 `＋ Add Star` 버튼 → 이름/메모 입력 (메모를 쓰면 타인정보 동의가 필요).
- **드래그**: 아무 별이나 마우스로 끌면 연결된 별들이 신축성 있게 따라옵니다.
- **확대/축소**: 마우스 휠. 배경을 끌면 이동(pan).
- **매칭 체험**: 내 별 클릭 → `초대 링크 만들기` → `상대가 수락했다고 가정 (데모)` → 그 사람의 인맥이 희미한 별로 확장됩니다.
- **소개 요청**: 희미한 별을 클릭 → `소개 요청하기`.
- **성능 데모**: 우측 상단 `성능 데모` → 300개 노드 + FPS 표시.

## 프라이버시 (중요)

타인에 대한 메모(`note`)는 **소유자에게만** 보입니다. 매칭된 상대조차 내 메모를 볼 수 없고,
2-hop으로 확장된 '희미한 별'에는 이름만 노출되고 메모는 절대 포함되지 않습니다.
이 불변식은 `DataProvider.test.ts`로 자동 검증됩니다.
